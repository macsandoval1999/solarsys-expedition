# solarsys-expedition
A website for those curious about our neighbors in the solar system.
Learn about the planets in our solar system by navigating through an interactive representation of it. 
Click on a planet to learn about it, and check out images pulled from NASAs database itself.
Check in daily to see NASAs Astronomical Picture of the Day.
Save your favorite pictures so you can see them whenever you want.

# Cool Features 
    - Interactive Solar System Representation
    - Planetary Data
    - Astronomical Images
